# 🐧 DevOps Linux Environment Setup Script

A comprehensive bash script to automate the setup of a complete DevOps development environment on Linux. This project demonstrates Linux system administration, bash scripting, and automation skills - fundamental for any DevOps engineer.

## 🎯 Overview

This script automates the tedious process of setting up a new Linux machine for DevOps work. Instead of manually installing and configuring each tool, you can run one script and get a fully configured environment with:

- Development tools (git, vim, curl, etc.)
- Container tools (Docker, Docker Compose)
- Orchestration tools (kubectl)
- Infrastructure as Code tools (Terraform)
- CI/CD tools (Jenkins)
- Custom configurations and aliases

## ✨ Features

- Multi-distribution support: Works on Ubuntu, Debian, RHEL, CentOS, Fedora
- Modular installation      : Install only what you need
- Interactive menu          : Choose components to install
- Smart detection           : Detects existing installations and skips reinstallation
- Comprehensive logging     : All actions logged for debugging
- Configuration management  : Sets up git, vim, bash with DevOps-friendly configs
- Error handling            : Graceful failure with clear error messages

## 📋 Prerequisites

- Linux machine (physical, VM, or cloud instance)
- sudo access
- Internet connection
- Basic understanding of terminal commands
